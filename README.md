## Запуск

Для запуска требуется [Node.js](https://nodejs.org/en/)

### Разработка
```
npm run dev
```
С режимом hot-reload

### Запуск линтера
```
npm run lint
```

### Запуск production сервера
```
npm run build
npm run start
```
